# Install from GitHub Source

## Quick Install (For Advanced Users)

If you want to install IP Guard directly from the GitHub repository without waiting for the Edge Store:

---

## Method 1: Download ZIP (Easiest)

### Step 1: Download
1. Go to: https://github.com/yaojeffrey/ipGuard
2. Click the green **"Code"** button
3. Click **"Download ZIP"**
4. Extract the ZIP file to a folder (e.g., `C:\Extensions\ipGuard`)

### Step 2: Install in Browser

**For Microsoft Edge:**
1. Open Edge and go to: `edge://extensions/`
2. Enable **"Developer mode"** (toggle in bottom-left)
3. Click **"Load unpacked"**
4. Browse to the extracted folder and select it
5. Done! The IP Guard icon appears in your toolbar

**For Google Chrome:**
1. Open Chrome and go to: `chrome://extensions/`
2. Enable **"Developer mode"** (toggle in top-right)
3. Click **"Load unpacked"**
4. Browse to the extracted folder and select it
5. Done! The IP Guard icon appears in your toolbar

---

## Method 2: Git Clone (For Developers)

### Step 1: Clone Repository
```bash
git clone https://github.com/yaojeffrey/ipGuard.git
cd ipGuard
```

### Step 2: Install in Browser
Follow the same browser installation steps as Method 1 (use the cloned folder)

## Updating Manual Installation

To update your manually installed extension:

1. **Download latest version** from GitHub (repeat Method 1)
2. **Delete old folder** (optional, or overwrite)
3. **Go to** `edge://extensions/` or `chrome://extensions/`
4. **Click reload icon** on the IP Guard extension card
5. Or **remove and re-add** the updated folder
